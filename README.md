# RecipeBook
It's a Xamarin.Forms project with using Sqlite Database. 

MAJOR FEATURES 

Register/Login, 
Adding Recipe with owners user name, 
Deleting Recipe

NUGET PACKAGES AND DEPENDENCIES 

NETStandard.Library,  
sqlite-net-pcl,  
Syncfusion.Xamarin.Expander,  
Xamarin.CommunityToolkit,  
Xamarin.Essentials,  
Xamarin.Forms
